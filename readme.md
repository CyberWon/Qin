# 秦的诞生

在运维领域，有着众多优秀的开源项目。

怎么才能把这些项目联系在一起，创造更大的价值？

历史给出了答案，大一统。

秦，应运而生。

# 说明

秦（Qin）是一款集成身份认证、租户接入和应用防护的Web网关。
